Recuva — небольшая бесплатная утилита для восстановления файлов случайно удаленных или потерянных в результате программного сбоя или ошибки. С ее помощью можно просканировать выбранный пользователем носитель информации.
После сканирования Recuva выдает список удаленных файлов. Каждый из этих файлов имеет собственные параметры: имя, тип, путь, вероятность восстановления, а также состояние (перезаписан и восстановлению не подлежит, частично перезаписан, полностью цел).
Специальная система фильтров позволяет определит ненужные файлы — необходимые файлы отмечаются флажками. Есть также функция предпросмотра, позволяющая определить, нужен ли пользователю данный файл. Среди дополнительных опций — возможность выбора режима отображения найденной информации — простым списком, или деревом, с учетом структуры каталогов.Язык интерфейса: multi (русский присутствует)
