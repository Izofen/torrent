###  Подробнее

Шаблон Horses для After Effects позволяет быстро и легко создавать собственные анимации лошадей. В комплекте пять реалистичных марионеток-лошадей разного цвета шерсти. С помощью простых в использовании элементов управления вы можете «запрограммировать» своих лошадей практически на все (сохранение вида профиля).
