В релиз-кандидат вошли следующие обновления стабильный glibc 2.14.1; стабильный релиз ядра  kernel 3.3.4; пре-релиз PulseAudio 2.0; CUPS  был обновлен до версии 1.5.2; KDE 4.8.2, и GNOME 3.4.1.
