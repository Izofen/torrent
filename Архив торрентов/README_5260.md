Java Runtime Environment - исполнительная среда созданная Sun Microsystems позволяющая запускать и использовать приложения написанные на языке программирования Java. Некоторые веб-страницы также используют Java. Одним из преимуществ языка Java является его кроссплатформенность, то есть приложения написанные на этом языке можно использовать на разных операционных системах где установлен Java Runtime Environment.Язык интерфейса: Multi/Русский
