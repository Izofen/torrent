###  Подробнее

Lost Memories - это проект, который имеет приятную винтажную кинематографическую атмосферу для того, чтобы показать ваши архивные фотографии или видео.
Проект предоставлен в двух версиях. Первая полностью редактируемая. Вторая, (Pre render) для более быстрого вывода видео-файла. Ни каких сторонних плагинов не требуется.Формат: AEP
