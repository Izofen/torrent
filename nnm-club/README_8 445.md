Rainlendar - cимпатичный календарик для рабочего стола с поддержкой списка задач на тот или иной день и возможностью переноса соответствующих настроек из Microsoft Outlook. Допускается изменение внешнего вида сменой скина, а также изменение уровня прозрачности календаря. Для управления программой возможно задание горячих клавиш. Поддерживаются плагины.Название программы: Rainlendar ProЯзык интерфейса: МультиРус.
