###  Подробнее

Молодой полицейский, работающий под прикрытием, сдан дважды. Сначала о нём сообщают уголовникам его же друзья из полиции, а затем начальство решает, что для успеха операции один из своих должен погибнуть. Главный герой оказывается в незавидной ситуации, из которой практически нет никакого выхода.
